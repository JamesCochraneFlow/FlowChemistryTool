# FlowChemistryTool

Tool to construct flow chemistry reactors and generate experimental write up/CSV data

For user demonstration see https://youtu.be/_JStf_xXjjM?si=JvYF5P_vlIsLDyjV

Download index.html, css.css, app.js, and presets.js

Put all 4 files together in a folder and then open index.html in browser of choice to use tool
